# Portfolio & Internship Match System

## Project Overview
This system helps students showcase their portfolios and get matched with internship opportunities based on skills, projects, and achievements. It is designed to make the internship application process easier for both students and companies.

---

## Frontend
- Built using **React.js**
- Features:
  - Interactive portfolio display
  - Internship listing and filtering
  - Responsive and user-friendly UI

---

## Database
- Tables include:
  - `Students`
  - `Portfolios`
  - `Internships`
  - `Applications`
- Supports queries for matching students to internships based on skills and eligibility
- ER Diagram and SQL schema are included in the `/database` folder

---

## How to Run Locally
1. Clone the repo:  
   ```bash
   git clone https://github.com/Nandini70594/Portfolio---Internship-Match-System.git

2. Navigate to frontend folder and install dependencies:

cd frontend
npm install


3. Start the frontend server:

npm start


4. Set up the database using the files in /database.

Folder Structure
/frontend    → React frontend
/database    → SQL schema, queries, ER diagram
/docs        → Project report, presentation, screenshots
